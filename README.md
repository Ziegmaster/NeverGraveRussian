# Русификатор для игры Never Grave
https://store.steampowered.com/app/2395770/Never_Grave_The_Witch_and_The_Curse/

Для установки русификатора необходимо скачать файлы NeverGraveRussian.dll и NeverGraveRussian.json со [страницы последнего релиза](https://github.com/Ziegmaster/NeverGraveRussian/releases/latest).
Файлы необходимо поместить в папку Mods после того как вы пропатчите игру через MelonLoader.
